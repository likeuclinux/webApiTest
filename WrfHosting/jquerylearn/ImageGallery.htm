Image Gallery

After having basic knowledge about writing plugin, now we will move ahead to create image gallery plugin. Suppose you have images on the web page and you want to maximize it when users click on image. So let’s start writing your plugin. User will pass image collection to our plugin and we will add functionality of click event. 

         $(document).ready(function () {
            $("img").imageGallery();
        });

Our plugin will get the collection of images and register the click event of each image. On the click of image, we will create copy of clicked image using clone() method. After setting the width, height, border etc. of image object, we will add image object into the page using prependTo(‘body’) statement. 

    var image = e.clone();
    .
    .
    . 
    image.prependTo('body').center();

Here is the complete code of the plugin 

    
    $.fn.imageGallery = function () {

        return this.each(function () {

            var e = $(this);
            var h = e.height();
            var w = e.width();

            e.height(75);
            e.width(75);

            e.click(function () {


                $("#gallery_popup_image").remove(); // remove last opened image, if any

                var image = e.clone(); // create duplicate object

                image.attr("id", "gallery_popup_image");
               
                image.css({
                            position:"absolute",
                            border:"6px solid #fffa7c"
                          });

                image.height(h);
                image.width(w);
                image.prependTo('body').center();

                image.click(function () {
                    $(this).remove();
                });
            });
        });

    };
